# Icons

このディレクトリには拡張機能のアイコン画像を配置します。

## 必要なアイコンサイズ

以下のサイズのPNG画像を作成してください：

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## デザインの推奨事項

- シンプルで認識しやすいデザイン
- Notion（データベース/ページ）とAI画像生成を組み合わせたイメージ
- 透明な背景を推奨
- 高コントラストで視認性の高いカラー

## アイコンがない場合

アイコンがなくても拡張機能は動作しますが、Chromeはデフォルトのアイコンを表示します。
